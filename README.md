the rule.

```
9:03-9:28a (no 5s, no 0s)

:: end of day ::
6:57p

1 - immediately
2 - immediately
3 - 1h delay
-----------------
::  if 7 sends before ::end of day::
4 - 2-2.5h delay
5 - 1.5-3h delay
6 - 1m delay
7 - 10-15m delay
-----------------
::  otherwise skip to ::start of day::
8 - immediately
9 - immediately

```
